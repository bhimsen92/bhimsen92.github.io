2020 3/4 In Review

  2020-08-26 Word Count: 329 words Read Time: 2 minutes

I always wanted to blog as far as I can remember but for some reason it never came to fruition. Probably because I am very lazy. But this time I have made my mind that I will blog every week no matter how silly the posts are. This gives me an opportunity to introspect about my life and also a nice way to improve my english language skills.

I don’t have to mention how good 2020 is for everyone :). I love travelling and I wanted to visit so many coutries this year but all that dream went up in smoke thanks to covid :(. I was lucky to visit Switzerland before the pandemic reared its ugly face. It was a fantastic trip, seeing snow caped mountains and sipping hot chocolate is the best feeling in the world. I hope someone will find a cure for covid as soon as possible so that we can all start living our life just like before.

On learning front, I wanted to concentrate more on system design, kubernetes, postgresql, react and understanding the concepts of distributed systems. I can’t say that I have achieved everything but now I know more about kubernetes and postgresql than ever before. I will write more about k8s and postgresql in the coming weeks.

I can’t recommend this course enough if you want to improve your system design skills. Though this is more oriented towards acing the interviews but for a beginner this is a good starting point. It will cover enough ground and you will also know where to look for if you want to learn more in the future.

Also Microsoft Azure Architectures Center is worth reading if you want to dig deeper into system design.

By the end of this year I would like to complete following tasks:

  • Complete reading Data Intensive Applications
  • Complete React course from React, Redux
  • Complete Redis In Action
  • Complete Kafka Definitive Guide
  • Complete Disitribute System Course on Courseera

Bye.
